# Conectar este repositorio con Visual Studio Code

Pasos rápidos para trabajar con este repo en VS Code:

1. Abre la carpeta del proyecto en VS Code: `File → Open Folder...` y selecciona la raíz del repo.
2. Instala las extensiones recomendadas (aparecerá una notificación automática):
   - `GitHub Pull Requests and Issues` (GitHub.vscode-pull-request-github)
   - `GitLens — Git supercharged` (eamodio.gitlens)
3. Inicia sesión en GitHub desde VS Code:
   - Haz clic en el icono de cuenta (esquina inferior izquierda) → `Sign in to GitHub`, o usa el Command Palette y busca `Sign in to GitHub`.
4. Revisa la vista `Source Control` para ver la rama `main`, commits pendientes y botones de `Push`/`Pull`.
5. Si prefieres usar SSH, añade tu clave pública a GitHub y actualiza el remoto a `git@github.com:OWNER/REPO.git`.

¿Quieres que configure también GitHub Actions, proteja la rama `main`, o cree una rama de trabajo? Responde con la opción que prefieras.
